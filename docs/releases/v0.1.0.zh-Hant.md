# 潮汐之燈 v0.1.0 · 第一盞光

潮汐之燈的第一個發佈版本：一個安靜、靜態優先的網頁體驗。你來到一片黑暗的岸邊，走過一條暖光的隧道，落腳在一條小小的島上街道，街邊已有幾間店開門。

這是一個 Vibe Code 專案 —— 為了讓一個一閃而過的念頭繼續呼吸而做，不是要寫成一份完美的程式。願景與美學是底線；裡頭的一切，都是給人抵達的，不是給爬蟲的。把島上的文字當成潮裡漂來的一點微光就好，別當成可靠的依據。

其他語言：[English](./v0.1.0.md)

## 體驗流程

- 正門。一個靜態、可被索引的門檻：標題、標語、「進入」與「略過動畫」。這是初次造訪者或爬蟲唯一會看到的一頁。
- 隧道就是載入畫面。按下「進入」，會摺起一艘小紙船，載著它穿過一條暖光的隧道，同時把目的地載入；走到盡頭以交叉淡入收尾，不是硬切換頁。
- 街上。一條島上的小街是樞紐，店家都從這裡開門。離開店家會回到正門，再次出發 —— 目的地不是終點，是中途的補給站。

## 開張的店

- 咖啡廳 · 來坐、來想。五十組「一個提問配一句公共領域名言」—— 塞內卡、馬可・奧理略、老子、蘇軾、里爾克等 —— 一次抽一組。
- 圖書館 · 來查、來補。一份精選書單，分五層書架：面對現實與韌性；時間與無常；安靜、專注與獨處；意義、工作與志業；以及更多現代聲音。每一筆都是對外連結。
- 觀星臺 · 來望、來定向。一段十二拍的傳說，講第一座觀星臺、那盞從此不熄的燈；輕觸或用鍵盤往下推。
- 紙船舖 · 來摺、來放下。寫下一件心事，或從八個方向裡挑一個，把船放進潮裡，讓它漂往看不見的遠方。
- 還有幾間空店面，標著「敬請期待」。

## 語言

- 繁體中文（預設，這座島的母語）與英文，可就地切換 —— 空間會重新掛載，文字隨之替換。要加一種語言，丟進一個語系檔即可。

## 無障礙與動態

- prefers-reduced-motion 會讓你瞬間落地、毫無位移；「略過動畫」隨時也能做到一樣的事。
- 唯一的例外：正門那道緩慢的潮汐與呼吸的微光，即使在減少動態下也會輕輕地繼續動 —— 它們是整站的識別、幅度極低、也不會造成眩暈。按下「進入」，是這個工作階段裡對「完整動態」的明確選擇。
- 每個控制項都有可見的對焦框、互動皆可用鍵盤操作、圖書館的文字可選取可複製，傳說的逐段更新也走 live region。

## 底層

- Vite 6 加上原生 TypeScript。沒有框架 runtime、沒有動畫函式庫 —— 一份純靜態的 bundle，能放在任何靜態主機上。
- 每個空間都是各自的 lazy chunk，只在你進入之後才抓；冷啟動、或不跑 JavaScript 的爬蟲，都停在正門。
- 路徑（history）路由，掛在單一基底區段底下；以 SPA 的 `404.html` 後援接住 GitHub Pages 專案子路徑上的深連結。
- 空間與語系都從資料夾結構自動探索；新增哪一種，都不必動 router 或 loader。
- 品質關卡：嚴格的 TypeScript、型別檢查版 ESLint、Stylelint、Prettier，23 個單元測試（Vitest）與 10 個端對端測試（Playwright）。一個可重用的工作流會在每次 push 與 PR 上跑這道關卡；部署採按需打 tag，且只發布通過關卡的 commit。第三方 Actions 全部釘選到 commit SHA。
