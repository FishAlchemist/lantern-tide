# 潮汐之燈 v0.3.0 · 角落的一枚小記

每一盞燈，都是在某個時刻點起的，也帶著那個時刻的印記。如今這座島，悄悄戴上了一枚 —— 角落裡，多了一個小小的版本記號，告訴你此刻手裡握著的，是哪一盞光。

這是一個 Vibe Code 專案 —— 為了讓一個一閃而過的念頭繼續呼吸而做，不是要寫成一份完美的程式。把島上的文字當成潮裡漂來的一點微光就好，別當成可靠的依據。煉光舖只是個讓心情放鬆的地方，不是專業的醫療場所。

## 角落的一枚小記

角落上，如今靜靜亮著一行淡淡的 `v0.3.0`。輕輕一點，它會微微打開：你正站著的這一版（它的提交碼），還有一條回到這一版記事的路 —— 也就是你正在讀的這一頁。不必特別在意它，只是替你留了一條線，通往這盞光是在哪裡點起的；哪天想循著走回去，它都還在。

## 燈下的小註

這還是一盞剛點起的光，憑著一個一閃而過的念頭亮著，也只趁它還亮著時亮著。這裡的一切都不是承諾 —— 房間、文字與細節，仍可能在每次造訪之間悄悄改變。能用上的就帶走，其餘的，讓它隨潮漂去。
