# Lantern Tide v0.3.0 — A mark in the corner

Every light is lit at some moment, and carries the marks of that moment. Now the island wears one, quietly: up in the corner sits a small version mark — which light you're holding, right now.

This is a Vibe Code project — made to let a passing idea keep breathing, not to be a flawless codebase. Treat its words as a small light drifting in on the tide, not a source to rely on. The Stillroom is a place to ease the mind, not medical care.

Also available in: [Traditional Chinese](./v0.3.0.zh-Hant.md)

## A mark in the corner

Up in the corner, a faint `v0.3.0` now keeps watch. Touch it and it opens a little: the exact build you're standing in — its commit — and a way back to the notes for this very version, the page you're reading now. Nothing you need to mind; just a thread back to where this particular light was lit, there if you ever care to follow it.

## A note under the lamp

This is still an early light, lit on a passing thought and kept burning only while it glows. Nothing here is a promise — rooms, words, and small details may still shift from one visit to the next. Take what helps you, and let the rest drift on with the tide.
